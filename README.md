# 디스코드 API
HQ 플러그인에서 디스코드 관련 기능을 사용하기 위해 넣어야 하는 필수 플러그인입니다.

## 다운로드
* 플러그인 다운로드는 [릴리즈](https://github.com/CosinePlugin/HQDiscordAPI/releases)에서 가능합니다.

## 디스코드
* 모든 문의, 질문, 개발 요청은 [디스코드](https://discord.gg/hUkaca9ZQu)에서 해주시길 바랍니다.

## 필수 플러그인
